Dischord
===

## 説明

Discord BotとしてMML(Music Macro Language)を提供します。

## 依存

- Node.js

## 使い方

### Botの起動

1. `DISCHORD_TOKEN`環境変数にDiscord Botのトークンを入力します。

2. 起動します。

    ```
    $ npm install
    $ npm run start
    ```

3. 🎉 Yes! Party time!! 🍣

### Botの使い方

デフォルトのプレフィックス(コマンド前に挿入すべき文字列)は`dc!`です。

- dc!help

    MMLの文法を表示します。

- dc!play

    MMLを音声としてコンパイルし、ファイルとして送信します。
